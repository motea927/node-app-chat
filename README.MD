## Mt chat
> 一款限時、隱密的聊天室

## Mt chat有哪些功能？

* 所有聊天紀錄均無紀錄，關閉瀏覽器即消失
* 可與朋友自行約定房間名稱，需輸入相同房間名稱才可聊天
* 新加入訪客無法回溯觀看加入房間前之訊息


## 使用技術、語言、套件及框架

* html、css、javascript、jquery、node.js
* node-modules:
    * express
	* moment
	* socket.io
